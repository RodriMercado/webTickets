🎟️ WebTickets — Ticket Sales System
--------------------------------------------------------------------------------
WebTickets is a web application developed in PHP using the CodeIgniter framework.
The project implements a ticket sales system for events with user management, an admin panel, and full CRUD operations.

🧑‍💻 Project developed as part of a university course

--------------------------------------------------------------------------------
🚀 Main Features

🔐 User system (registration, login, and roles)
🧾 Ticket management (create, edit, delete, list)
🛒 Ticket purchasing module (selection and cart)
📊 Full CRUD operations implemented with CodeIgniter
🎨 Responsive design using Bootstrap 5, HTML, and CSS
🗄️ MySQL database with included script (Script_webtickets.sql)
